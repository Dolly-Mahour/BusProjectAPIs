# Bus Project APIs

A RESTful backend API application built using Node.js, Express.js, and MongoDB.  
This project provides APIs for managing buses, users, authentication, cities, and places for a bus booking/travel platform.

## 🚀 Features

- User Authentication using JWT
- Login & Signup APIs
- CRUD operations for buses and places
- MongoDB database integration
- RESTful API architecture
- Express middleware implementation
- Secure password handling
- Organized backend folder structure

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcrypt.js
- dotenv
